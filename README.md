
 Fanorona 

**Fanorona** est un jeu de stratégie combinatoire abstrait traditionnel de Madagascar, vieux de plusieurs siècles, souvent qualifié d’échecs malgaches. Il se joue à deux avec 22 pions chacun sur un plateau composé de lignes et d’intersections. Le but est de capturer tous les pions de l’adversaire par **percussion** ou **aspiration**.

Ce projet consiste en une implémentation mobile moderne du jeu, développée avec **React Native** et **Expo (SDK 54)**.

---

##  Objectifs du projet

* Reproduire fidèlement les règles du Fanorona
* Offrir une interface intuitive et fluide
* Permettre le jeu local (2 joueurs sur le même appareil)
* Permettre de jouer avec un bot avec 3 difficulté (easy-normal-hard)
  

## 📦 Installation

```bash
git clone https://github.com/ton-username/fanorona-app.git
cd fanorona-app
npm install
npx expo start
```
